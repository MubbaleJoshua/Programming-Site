HTML Basics

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create web pages. HTML describes the structure of a web page using elements.

Basic HTML Page

<!DOCTYPE html> <html> <head> <title>My Page</title> </head> <body> <h1>Hello World</h1> <p>My first web page</p> </body> </html>

Important HTML Elements

  • h1 – heading
  • p – paragraph
  • a – hyperlink
  • img – image
  • div – container
