Welcome to Programming Academy

Programming is the process of creating instructions that computers follow to perform tasks. These instructions are written using programming languages.

Learning programming allows you to build:

  • Websites
  • Mobile applications
  • Software systems
  • Games
  • Artificial intelligence

Learning Path

  1. Learn HTML (structure of web pages)
  2. Learn CSS (design and layout)
  3. Learn JavaScript (interactivity)
  4. Learn a general programming language such as C or Python
  5. Build projects
